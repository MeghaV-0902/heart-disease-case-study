# Heart Disease Prediction

## Overview
This project builds a machine learning model to predict the presence of heart disease using clinical parameters.

---

## Problem Statement
Early detection of cardiovascular risk is critical for preventive healthcare. The goal is to predict heart disease presence using patient health indicators.

---

## Objective
- Perform Exploratory Data Analysis  
- Engineer relevant features  
- Compare multiple ML models  
- Select best model based on evaluation metrics

---

## Dataset

- 180 observations  
- 19 total columns  
- 18 input features  
- Target variable: `heart_disease_present`

The dataset includes demographic, clinical, and stress test-related measurements used to predict cardiovascular risk.

Key numerical features:
- age
- resting_blood_pressure
- serum_cholesterol_mg_per_dl
- max_heart_rate_achieved
- oldpeak_eq_st_depression
- num_major_vessels
